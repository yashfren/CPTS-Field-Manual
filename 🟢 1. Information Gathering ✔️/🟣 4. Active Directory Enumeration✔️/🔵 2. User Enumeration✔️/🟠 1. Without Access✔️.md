## Using CrackMapExec --users Flag

###### We can use CrackMapExec with the --users flag. This is a useful tool that will also show the badpwdcount (invalid login attempts), so we can remove any accounts from our list that are close to the lockout threshold. It also shows the baddpwdtime, which is the date and time of the last bad password attempt, so we can see how close an account is to having its badpwdcount reset.

```bash
Yashfren@htb[/htb]$ crackmapexec smb 172.16.5.5 --users
```
